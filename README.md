# react-redux-example

> An example of React Redux async application

## Install

```sh
git clone https://github.com/andrepolischuk/react-redux-example
cd react-redux-example
npm install
```

## Run

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```
